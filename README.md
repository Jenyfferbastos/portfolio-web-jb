## Running locally

Clone the project

```bash
  git clone https://github.com/Jenyfferbastos/portfolio-web-jb.git
```
Enter the project directory

```bash
  cd portfolio-web-jb
```

Start the server

```bash
    Open your terminal and type the command npm run dev
```
